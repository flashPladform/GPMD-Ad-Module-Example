Classpublic class AdWrapper
InheritanceAdWrapper flash.display.Sprite
Implements IAdModule

Данный объект несет отвественность за воспроизведение рекламы. В качестве базовой идеи взят VPAID с некоторыми модификациями, таких как: - убрана функция запуска рекламы startAd - объект отправляет больльше событий показа рекламы
